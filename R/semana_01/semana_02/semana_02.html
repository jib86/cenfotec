Paso 1

Leer y escribir CSV 

# 1. Cargar paquete (instalar si es primera vez)
# install.packages("readr")
library(readr)

# 2. R trae datos de prueba. Usaremos 'mtcars' (Motor Trend Cars).
# write_csv() exporta esa tabla de R a un archivo real en tu compu.
write_csv(mtcars, "vehiculos.csv")

# 3. read_csv("ruta") -> lee un archivo y lo mete a R
datos_autos <- read_csv("vehiculos.csv")

# 4. Ver el resultado
print(datos_autos)

¿Qué hace cada código?

read_csv("archivo.csv"): Lee un archivo guardado en tu computadora y lo mete adentro de R para que puedas analizarlo.

write_csv(tabla, "archivo.csv"): Hace lo contrario. Toma los datos que tienes en R y los guarda en tu computadora como un archivo Excel/CSV.

Regla de Oro de los Paquetes

install.packages() es como comprar una cafetera: lo haces una sola vez en tu vida.
library() es como enchufarla: lo debes correr cada vez que abres RStudio.

Explicación Mental

str(tabla): ("Structure") Sirve para ver qué hay adentro de tu tabla. Te dice rápidamente si una columna tiene números o tiene texto. Es vital hacerlo siempre.

El símbolo $: Es la llave para entrar a tu tabla. Escribes tabla$columna para decirle a R exactamente con cuál columna quieres trabajar.

Crear columnas: Si escribes tabla$nueva <- tabla$vieja * 2, R va fila por fila, multiplica el número y guarda el resultado en una columna totalmente nueva.

Troubleshooting:

¿Te sale error de "non-numeric argument to binary operator"?
Intentaste multiplicar o sumar una columna que R cree que es texto (`character`). Usa `str()` para confirmarlo.

Paso 3

dplyr: filter() y select() 

library(dplyr)

# Usaremos directamente mtcars de ahora en adelante
# Transformemos la variable 'am' (transmisión) a texto para leerla fácil
datos <- mtcars |>
  mutate(Transmision = ifelse(am == 0, "Automático", "Manual"))

# 1. filter(): subconjunto de filas (Solo Automáticos)
autos_automaticos <- filter(datos, Transmision == "Automático")

# 2. pipe |> : encadena pasos lógicos sin crear variables nuevas
resultado <- datos |>
  filter(hp > 150) |>       # Solo autos con más de 150 caballos de fuerza
  select(mpg, hp, Transmision)  # Retenemos 3 columnas
  
print(resultado)

El Pipe del Data Science

Usamos ifelse() para crear textos legibles a partir de una variable numérica de ceros y unos.

filter(): Filtra las filas. Mantiene solo donde la condición sea `TRUE`.

select(): Selecciona o descarta columnas por nombre.

|> (El Pipe): Toma la tabla que tiene a su izquierda y la "conecta" como entrada del código a su derecha. Se lee cómo "Y entonces...".

Troubleshooting:

¿Error "could not find function '|>'" o "filter"?
Probablemente olvidaste correr la línea `library(dplyr)`.

¿Error "object 'autos' not found"?
R distingue mayúsculas de minúsculas. 'autos' y 'Autos' son cosas totalmente distintas.

Paso 4

Agregaciones: group_by() 

# mutate() crea o modifica columnas, como $ pero integrable al pipe
datos <- datos |>
  mutate(Peso_kg = wt * 453.59) # wt está en miles de libras

# El combo letal: group_by + summarise
por_cilindros <- datos |>
  group_by(cyl) |>
  summarise(
    promedio_caballos = mean(hp),
    peso_medio_kg = mean(Peso_kg)
  )

print(por_cilindros)

Ensamblaje

mutate(nueva = calculo): Fabrica una columna nueva de forma rápida y la pega al final de tu tabla, sin borrar las que ya tenías.

group_by(columna): Agrupa tus datos por categorías invisibles (por ej: separa mentalmente los autos de 4, 6 y 8 cilindros). No hace nada visual por sí solo.

summarise(promedio = mean(x)): Toma esos grupos invisibles y los "aplasta", calculando un resumen (como el promedio, la suma o el total) para cada grupo independiente.

Paso 5

Vistazo Rápido: summary() 

# summary() da mínimo, cuartiles, media, máximo de cada columna
summary(datos)

# Medidas individuales
media_consumo   <- mean(datos$mpg)
mediana_consumo <- median(datos$mpg)
var_consumo     <- var(datos$mpg)

cat("Media Mpg:", media_consumo, "| Mediana Mpg:", mediana_consumo)

Exploración inicial

summary(tabla): Te da un resumen estadístico automático de toda tu tabla: el valor más pequeño, el más grande y el promedio. ¡Es oro puro para conocer tus datos en un segundo!

mean(numero): Calcula el promedio matemático normal.

median(numero): Calcula la mediana (el número que queda exactamente en el centro si los ordenas de menor a mayor).

var(numero): Calcula la varianza (qué tan dispersos o alejados están los datos entre sí).

Diagnóstico visual

table(columna): Cuenta cosas. Te dice exactamente cuántos autos hay de 4 cilindros, cuántos de 6, etc.

barplot(tabla): Dibuja un gráfico de barras. Es perfecto para comparar cantidades de distintas categorías.

hist(columna_numerica): Dibuja un histograma. Es como un escáner para tus números: te muestra visualmente si la mayoría de los autos consumen poca gasolina, mucha, o si están en el medio.

Estudio de Caso

Análisis de una Flota Vehicular

El Problema de Negocio:

Una aseguradora de vehículos necesita analizar una flota para ajustar sus pólizas y nos hace dos preguntas:
1. ¿Cuáles son los 3 vehículos que más consumen gasolina?
2. ¿Existe una diferencia clara en potencia (caballos de fuerza) entre autos de 4, 6 y 8 cilindros?

Usaremos el dataset mtcars. R guarda los nombres de los carros como "nombres de filas", así que primero ejecutaremos este código para extraerlos a una columna formal llamada Modelo: 

library(dplyr)
autos <- mtcars |> 
  mutate(Modelo = rownames(mtcars)) |> 
  select(Modelo, mpg, cyl, hp) # Nos quedamos con lo esencial

Estudio de Caso

Pasos 5 a 7 

# Pregunta 1: ¿Cuáles son los 3 vehículos que más gasolina consumen?
# Nota: Mayor mpg = rinde MÁS millas por galón (consume menos).
# Así que "más consumen" = Menor mpg
peores_consumos <- autos |>
  arrange(mpg) |>        # Ordenar de menor a mayor mpg
  head(3)                # Tomar los primeros 3

print(peores_consumos)

Ordenar y Top

arrange(columna): Ordena todas las filas de la tabla usando esa columna específica. Por defecto lo hace del número más pequeño al más grande.

head(3): Le pedimos a R que corte el resultado gigante y solo nos muestre los primeros 3 autos de esa lista (los que rinden menos millas).

Comando ¿Para qué sirve de forma simple?
read_csv() Mete un archivo Excel/CSV adentro de R.
str() Revisa qué hay dentro de tu tabla (números vs texto).
